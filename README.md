# Sales Performance Dashboard (Power BI)

This project presents a Sales Performance Dashboard built using Power BI to analyze key business metrics.

## Project Overview
The dashboard provides insights into total sales, total profit, profit margin, and quantity across product categories, sub-categories, regions, and time.

## Key Features
- KPI analysis for Sales, Profit, Profit Margin %, and Quantity
- Category and Sub-Category performance analysis
- Regional sales performance visualization
- Monthly sales trend analysis
- Interactive filters for Region, Category, and Order Date

## Tools Used
- Power BI
- DAX
- Power Query
- Excel

## Files Included
- Power BI dashboard file (.pbix)
- Dataset used for analysis
